# First Feature Flag

Ship a feature flag in under 5 minutes.

## 1. Create a flag in the dashboard

Open **Flags** in the sidebar and click **Create flag**.

- **Name**: `New Checkout`
- **Key**: `new-checkout`
- **Type**: Boolean
- **Default value**: `false`

Click create, then toggle the flag **on** for your production channel.

## 2. Install the SDK

```bash
npm install @appdispatch/openfeature-provider @openfeature/react-sdk
```

## 3. Initialize the provider

```jsx
import { OpenFeature, OpenFeatureProvider } from '@openfeature/react-sdk'
import { DispatchProvider } from '@appdispatch/openfeature-provider'

OpenFeature.setProvider(
  new DispatchProvider({
    baseUrl: 'https://api.appdispatch.com',
    projectSlug: 'your-project',
    apiKey: 'YOUR_API_KEY',
    channel: 'production',
  })
)

function App() {
  return (
    <OpenFeatureProvider>
      <YourApp />
    </OpenFeatureProvider>
  )
}
```

## 4. Use the flag

```jsx
import { useBooleanFlagValue } from '@openfeature/react-sdk'

function CheckoutButton() {
  const newCheckout = useBooleanFlagValue('new-checkout', false)

  if (newCheckout) {
    return <NewCheckoutFlow />
  }
  return <LegacyCheckout />
}
```

## 5. Toggle it

Go back to the dashboard, toggle the flag off. Reload your app — the old checkout appears.

Toggle it on — the new checkout appears. No deploy needed.
